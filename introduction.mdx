## What is AdRecon?

AdRecon is a SaaS ad intelligence platform that gives you direct access to a curated database of **90,000+ proven profitable affiliate ads** across the largest affiliate networks in the world ready for you to **copy and paste.**

Whether you're a media buyer, affiliate marketer, or creative strategist, AdRecon helps you discover what's working in real-time across:

- **ClickBank**
- **Digistore24**
- **BuyGoods**
- **MaxWeb**
- **Many Others**

## Key Capabilities

<AccordionGroup>
  <Accordion icon="rss" title="Live Ad Feed">
    Browse a continuously updated feed of active ads with normalized metadata including:

    - Creative images and landing page URLs
    - Network classification (ClickBank, Digistore24, BuyGoods, MaxWeb, Other)
    - Taxonomy classification (Health, Wealth, BizOpp, Relationship, Survival, Other)
    - Affiliate network metadata and product identifiers
  </Accordion>
  <Accordion icon="bookmark" title="Save & Organize Ads">
    Build your swipe file with:

    - One-click save for any creative
    - Project-based organization for campaigns or research
    - Personal saved ad library with instant filtering
    - Per-user isolation with row-level security
  </Accordion>
  <Accordion icon="download" title="Landing Page Ripper">
    Capture complete landing pages for offline analysis:

    - Self-contained HTML snapshots using headless Chromium + SingleFile
    - Bundled assets (CSS, JS, images, fonts, media) in a single ZIP download
    - Auto-scroll to trigger lazy-loaded content
    - SSRF protection and rate limiting (10 captures per 15 minutes)
  </Accordion>
  <Accordion icon="globe" title="Multi-Network Coverage">
    AdRecon normalizes data from multiple affiliate networks into a unified feed:

    - Consistent schema across all networks
    - High-quality media URL prioritization
    - Automatic classification and tagging
  </Accordion>
</AccordionGroup>

## Data Quality

AdRecon prioritizes high-quality media URLs and normalizes raw affiliate network data into a clean, consistent schema:

- **Normalized View**: `ads_feed_v2` provides a unified interface across all networks
- **Media Prioritization**: Automatic selection of the highest quality creative assets
- **Fixed Taxonomy**: Standardized categories across all ad sources
- **Active Monitoring**: Continuous data refresh from affiliate network sources
