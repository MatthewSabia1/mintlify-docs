## Overview

Projects let you create custom folders to organize your saved ads. Think of them as research binders — one for each campaign, niche, or client.

## Why Use Projects?

Projects help you:

- **Organize campaigns** — Group ads by offer, funnel, or promo
- **Niche research** — Collect ads for specific verticals
- **Client work** — Separate ads by client or agency project
- **Swipe files** — Build creative inspiration libraries
- **A/B testing** — Compare ad variations side-by-side

## Creating Projects

<Steps>
  <Step title="Switch to Projects Tab">
    Click **Projects** in the main navigation
  </Step>
  <Step title="Enter Project Name">
    Type a name in the input field (left sidebar)
  </Step>
  <Step title="Click Create">
    Press the **\+** button or hit Enter
  </Step>
</Steps>

<Info>
  Project names are case-insensitive and must be unique. You cannot create a project named "Saved" (reserved name).
</Info>

### Naming Best Practices

- **Be specific** — "Keto Supplement Launch Q1" vs. "Health Ads"
- **Include dates** — "Black Friday 2025 Promos"
- **Use client codes** — "ACME-001 Product Research"
- **Avoid duplicates** — System will reject duplicate names (case-insensitive)

## Adding Ads to Projects

Every saved ad can belong to multiple projects:

<Steps>
  <Step title="Save the Ad">
    Bookmark the ad from the feed (it must be saved first)
  </Step>
  <Step title="Open Save Modal">
    Click **Manage Save** in the ad detail modal
  </Step>
  <Step title="Select Projects">
    Check the boxes for all projects where this ad belongs
  </Step>
  <Step title="Confirm">
    Click **Save** to apply changes
  </Step>
</Steps>

<Note>
  An ad can be in zero or many projects. Removing from all projects keeps it in your Saved collection.
</Note>

## Viewing Project Ads

### Switching Projects

1. Click **Projects** tab
2. Select a project from the left sidebar
3. The feed updates to show only ads in that project
4. Filtered count appears in the project header

### Project Sidebar

The sidebar shows:

- **Saved** — Special folder for all saved ads not assigned to projects
- **Your Projects** — Listed by most recently updated
- **Active indicator** — Highlighted when selected
- **Quick actions** — Rename and delete buttons (visible on hover/tap)

## Managing Projects

### Renaming Projects

<Steps>
  <Step title="Hover Over Project">
    On desktop, hover to reveal action buttons
  </Step>
  <Step title="Click Edit Icon">
    Click the pencil icon
  </Step>
  <Step title="Type New Name">
    The project name becomes an editable field
  </Step>
  <Step title="Confirm or Cancel">
    Click the checkmark to save or X to cancel
  </Step>
</Steps>

<Info>
  On mobile, action buttons are always visible. Renaming validates against existing project names.
</Info>

### Deleting Projects

<Warning>
  Deleting a project removes all ad-to-project links but does NOT delete the ads. Ads remain in your Saved collection.
</Warning>

1. Click the trash icon next to the project name
2. Confirm the deletion in the dialog prompt
3. The project and all its ad associations are removed
4. If you were viewing that project, the feed switches to Saved

## Removing Ads from Projects

Two ways to remove an ad from a project:

### Method 1: Save Modal

1. Open the ad detail modal
2. Click **Manage Save**
3. Uncheck the project(s) to remove
4. Click **Save**

### Method 2: Project View

1. Navigate to the project
2. Find the ad in the feed
3. Click the trash icon (in Projects scope, this removes from current project only)

<Note>
  The trash icon behavior changes based on scope:

  - **All Ads** — No trash icon (ads can't be deleted)
  - **Saved** — Removes from entire collection
  - **Projects** — Removes from current project only
</Note>

## Project Workflow Example

Real-world scenario:

<Steps>
  <Step title="Create Project">
    New project: "Summer 2025 Weight Loss Campaign"
  </Step>
  <Step title="Browse & Save">
    Find 20 health ads in the feed, quick-save them
  </Step>
  <Step title="Assign to Project">
    Open each saved ad, check the project box
  </Step>
  <Step title="Filter by Project">
    Switch to Projects tab, select your campaign folder
  </Step>
  <Step title="Analyze">
    Review only the 20 relevant ads, apply filters as needed
  </Step>
</Steps>

## Multi-Project Ads

Ads can belong to several projects simultaneously:

- **Cross-campaign reuse** — Same creative in multiple funnels
- **Niche overlap** — Health + BizOpp hybrid offers
- **Client sharing** — One ad relevant to multiple clients

### Viewing Multi-Project Status

When viewing an ad:

- The modal shows "Saved · X projects"
- All assigned projects are checkboxed in the save modal
- Removing from one project leaves others intact

## System Folder: Saved

The **Saved** folder is a special system project:

- **Cannot be deleted** — Always available
- **Shows unassigned ads** — Displays all saved ads not in custom projects
- **No rename allowed** — Name is reserved
- **Acts as inbox** — Default landing for newly saved ads

<Info>
  Think of "Saved" as your inbox. Move ads to custom projects to organize them, or leave them in Saved for quick access.
</Info>
